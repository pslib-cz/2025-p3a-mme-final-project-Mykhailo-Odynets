# 3D Printed Clock Chassis

Funkční designový obal pro nástěnné hodiny. Cílem bylo vytvořit esteticky zajímavé šasi, které slouží jako „exoskeleton“ pro běžně dostupný hodinový mechanismus.

- Kompatibilita: Uzpůsobeno pro zasunutí klasického quartz strojku.
- Design: Inspirováno organickými strukturami (vycházející z konceptu Kletka), které minimalizují spotřebu materiálu při zachování pevnosti.
- Post-proces: Navrženo tak, aby po vytištění nevyžadovalo složité úpravy – stačí vložit strojek a zavěsit.
